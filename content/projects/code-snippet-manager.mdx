## Overview

A self-hosted snippet manager for storing and retrieving reusable code snippets. Think GitHub Gist but private, fast, and fully under your control.

## Features

- Syntax highlighting for 100+ languages (Shiki)
- Tag-based organisation and fuzzy search
- One-click copy to clipboard
- Public/private visibility per snippet
- Markdown description support
- Export to Gist

## Tech Stack

- **Framework**: Next.js (App Router)
- **Database**: SQLite via Prisma
- **Syntax Highlighting**: Shiki
- **Auth**: Auth.js (next-auth)
- **Styling**: Tailwind CSS

## Design Philosophy

I kept the stack as simple as possible — SQLite means zero infrastructure, a single binary database file, and instant deploys anywhere. For a personal tool, this is more than sufficient and makes backups trivially simple (`cp snippets.db backup.db`).
