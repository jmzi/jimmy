## Overview

This project is a full-stack web application demonstrating modern React patterns with Next.js App Router, server components, and Tailwind CSS for styling.

## Features

- Server-side rendering with App Router
- MDX content management
- Contact form with email integration
- Responsive design

## Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Deployment**: Vercel
