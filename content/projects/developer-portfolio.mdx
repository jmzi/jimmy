## Overview

The portfolio you're looking at right now. Built with a focus on simplicity, performance, and ease of content updates without a CMS.

## Features

- MDX-powered blog and project pages
- Responsive grid with stagger animations
- Dark mode ready
- Static generation for all content pages
- Contact form with server-side email handling

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Content**: MDX with custom frontmatter parser
- **Styling**: Tailwind CSS v4
- **Fonts**: Geist Sans / Geist Mono
- **Deployment**: Vercel

## Content Management

All content lives in `/content/blogs` and `/content/projects` as `.mdx` files. Adding a new post or project is as simple as creating a new file — no CMS, no database, no API keys needed.

The frontmatter parser handles `title`, `date`, `description`, `tags`, `image`, and `url` fields, which are surfaced automatically in the listing and detail pages.
